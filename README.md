# Resident-Evil-7-Biohazard-web
大學時期很喜歡惡靈古堡7而製作的網站，使用語言包含HTML與CSS。網站內容包含劇情簡述、武器與道具介紹、開發背景、DLC資料、外界評價等，並輔以圖片與影片呈現，讓使用者能更全面地了解本作的世界觀與魅力。
